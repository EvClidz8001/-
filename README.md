# 🌸 Flowers of Krasnodar — доставка цветов по Краснодару

[![GitHub Pages](https://img.shields.io/badge/Hosting-GitHub%20Pages-blue)](https://pages.github.com/)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)

Современный адаптивный сайт для заказа цветов с доставкой по **Краснодару**. Быстрый, удобный, с фокусом на продажи.

![Превью сайта](screenshot.png) <!-- Если добавите скриншот -->

## 🚀 Демо

Живая версия: [https://ваш_логин.github.io/название_репозитория](https://ваш_логин.github.io/название_репозитория)

## ✨ Особенности

- 📱 Адаптивный дизайн (мобильные, планшеты, ПК)
- 🏷️ Каталог с фильтрацией по типам цветов
- 🛒 Корзина с подсчётом суммы
- 📍 Интеграция с картами для зон доставки по Краснодару
- 📞 Форма обратной связи
- ⚡ Быстрая загрузка (оптимизированные изображения)

## 🛠️ Технологии

- **HTML5** — семантическая вёрстка
- **CSS3** (Flexbox/Grid, анимации, адаптив)
- **JavaScript** (чистый, без фреймворков)
- **LocalStorage** — сохранение корзины
- **GitHub Pages** — бесплатный хостинг
